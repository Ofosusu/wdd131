CareBridge Ghana Site Plan

WDD 131 Dynamic Web Fundamentals

Site Name

CareBridge Ghana

This name was chosen because it represents connecting caregivers with individuals and organizations that need quality care services. The website focuses on caregiving education, training, and employment opportunities.

Optional Domain: carebridgeghana.org

Site Purpose

The purpose of this website is to provide information about caregiving careers, caregiver training opportunities, professional certifications, and employment pathways. Visitors will be able to learn about caregiving services and contact the organization for support.

Scenarios

  • What qualifications do I need to become a professional caregiver?
  • How can I find caregiving job opportunities in Europe?
  • Are there caregiving training programs available in Ghana?
  • How can I contact the organization for more information?

Color Schema

Primary Color - #003366

Dark Blue will be used for headings, navigation, and important content areas.

Secondary Color - #FFD700

Gold will be used for buttons, accents, highlights, and call-to-action elements.

Typography

Poppins will be used for all headings.

Open Sans will be used for body text because it is easy to read on desktop and mobile devices.
